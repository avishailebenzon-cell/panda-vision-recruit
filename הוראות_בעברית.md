# הוראות פרסום ל Render ו Vercel - בעברית

## מצב כרגע ✅

הקוד שלך כבר עלה ל GitHub:
https://github.com/avishailebenzon-cell/panda-vision-recruit

זה כל מה שצריך! עכשיו פשוט צריך להפעיל את זה בענן.

## שלב 1️⃣: Render Backend (15 דקות)

### מה אתה צריך לעשות:

**1. הכנס ל Render:**
https://render.com

**2. לחץ "Sign up with GitHub"**
- בחר את המשתמש GitHub שלך

**3. לחץ "+ New" → "PostgreSQL"**

**4. מלא את הטופס:**
```
Name: panda-vision-recruit-db
Database: panda_vision
Username: postgres
Password: (Render יוצר אוטומטי)
Region: Northern Europe
```

**5. לחץ "Create Database"**
⏳ חכה עד שהסטטוס יהיה ירוק (5-10 דקות)

**6. כשסיימת - copy את ה"Internal Database URL"**
זה דומה לזה:
```
postgresql://postgres:PASSWORD@dpg-xxxxx.postgres.render.com:5432/panda_vision
```
**שמור את זה בטקסט כי תצטרך אותו בעוד שנייה!**

---

### עכשיו תנו את ה Backend:

**7. חזור לדשבורד Render, לחץ "+ New" → "Web Service"**

**8. בחר את ה repository:**
- panda-vision-recruit
- לחץ Connect

**9. מלא את הטפס:**
```
Name: panda-vision-recruit-api
Environment: Docker
Root Directory: .
Branch: main
Region: Northern Europe (זהה ל database)
```

**10. לחץ "Create Web Service"**

**11. כשנפתח, לחץ על "Environment"**

**12. הוסף את המשתנים האלה (one by one):**

```
DATABASE_URL = 
  (הערך שש'מרת בשלב 6 למעלה)

SECRET_KEY = 
  sk-ant-test12345randomstring

DEBUG = 
  false

APP_VERSION = 
  1.0.0
```

**13. לחץ "Deploy"** ⏳ חכה 2-5 דקות

**14. כשתראה "Your service is live" - בדוק:**
```bash
curl https://panda-vision-recruit-api.onrender.com/health
```

אם חזר משהו עם `"status":"healthy"` - יפה! ✅

**שמור את הכתובת:** `https://panda-vision-recruit-api.onrender.com`

## שלב 2️⃣: Vercel Frontend (10 דקות)

### מה אתה צריך לעשות:

**1. הכנס ל Vercel:**
https://vercel.com

**2. לחץ "Sign up with GitHub"**

**3. לחץ "Add New" → "Project"**

**4. בחר את ה repository:**
- panda-vision-recruit
- לחץ Import

**5. מה שנפתח:**
- Root Directory: `frontend`
- כל השאר זה טוב אוטומטי

**6. לחץ "Environment Variables"**

**7. הוסף משתנה אחד:**
```
VITE_API_URL = 
  https://panda-vision-recruit-api.onrender.com
```

(בדיוק הכתובת של Render שש'מרת למעלה)

**8. לחץ "Deploy"** ⏳ חכה דקה-שתיים

## אם משהו לא עבד?

### "Backend לא עלה"
- בדוק את ה logs של Render
- בדוק שהכנסת DATABASE_URL נכון
- פעם שנייה Copy את זה מה Database שיצרת

### "Frontend לא מחבר ל backend"
- בדוק בברראוזר - F12 → Console
- בדוק שהכנסת VITE_API_URL נכון ב Vercel

### "לא יודע מה לעשות בשלב מסוים"
- זה בסדר! עשה מה שאתה יכול
- בדוק את הסקרינשוט או וידאו של Render/Vercel אם יש
